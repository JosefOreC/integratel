# Integratel BI + AI

## Objetivo

Desarrollar una aplicación web para Integratel Perú que permita visualizar indicadores de negocio y predecir el riesgo de abandono de clientes (Churn).

La aplicación es un MVP para demostración académica.

No es una página corporativa.

## Usuarios

- Gerencia
- Analistas BI
- Atención al Cliente

## Módulos

- Dashboard
- Clientes
- Predicción
- Reportes
- Chat IA

## Alcance

La IA ya existe.

El modelo XGBoost ya fue entrenado.

La aplicación únicamente realiza inferencias.

No implementar entrenamiento.

No implementar pipelines ETL.

No implementar Data Warehouse real.

Solo demostrar conceptualmente su uso.

## Prioridades

1. Predicción funcionando
2. Dashboard profesional
3. Gestión de clientes
4. Reportes
5. Chat IA

## Objetivo de UX

Interfaz limpia.

Fácil de entender.

Máximo tres clics para cualquier acción.

Toda la información importante visible inmediatamente.

No sobrecargar pantallas.
